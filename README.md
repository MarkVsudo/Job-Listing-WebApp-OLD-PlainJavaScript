# A Job Board full-stack project

**Tech Stack (MVC software design pattern):** MySQL; EJS, JavaScript, CSS; Express.js

**For running the project, execute these two commands:**

1. 🏗 Download all dependencies
   ```
   npm i
   ```
2. 🏁 Run project
   ```
   nodemon app
   ```
